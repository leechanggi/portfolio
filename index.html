웹표준을 준수하고 반응형 웹을 좋아하는 웹 퍼블리셔 입니다 :)

#Experience

한국직업전문학교에서 스마트컨텐츠 UI/UX 퍼블리싱 수업을 수료하였고,
반응형 웹 제작에 관심이 많습니다.
귀로 들은 이야기보다 눈으로 본 것을 잘 기억하고,
본 것보다 직접 해본 것을 훨씬 잘 기억합니다.
다양한 경험이 중요하다고 생각합니다.

#Empathy

다양한 콘텐츠를 통한 공감과 소통은 더욱 중요해지고 있습니다.
저는 공감하고 소통하는 퍼블리셔가 되려고 노력합니다.
클라이언트와 사용자의 니즈가 충족된 결과물이 성공한 프로젝트이기 때문입니다.

#Together

디자이너, 개발자와 함께 일하며 원활한 협업을 하기위해 노력하겠습니다.
열정과 노력을 가지고, 오랫동안 함께 성장하고 싶은 퍼블리셔가 되고 싶습니다.

  • HTML5 사용 / 웹 표준에 준수한 마크업 / 유효성에 적합한 구조 설계
  • CSS3 사용 / 반응형 레이아웃
  • SASS(scss) 사용한 프리프로세싱 가능
  • 자바스크립트의 기초 문법 이해
  • 동적인 기능 구현 가능 / jQuery 플러그인 사용 / 오픈소스 활용
  • Photoshop의 다양한 툴 사용 가능
  • etc. 한글, Excel, Power Point 등 사용 가능 / 각 직군과 협업
